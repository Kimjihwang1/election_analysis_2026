# 투표소별 당일 투표 수요 예측 모델
(Election Day Vote Demand Prediction)

## 1. 프로젝트 배경 및 목적

- **배경:** 2026년 6.3 지방선거에서 선관위는 사전투표율 상승과 잔여 투표용지 관리 이슈를 이유로 본투표용지를 유권자 수의 50% 수준으로 일괄 배정하는 방식을 처음 적용했습니다.
- **문제점:** 전국 평균 사전투표율은 상승 추세이나, 읍면동 등 세부 지역별 사전투표율 편차는 매우 큽니다. 전체 평균에 맞춘 일괄 배정 방식은 사전투표율이 낮고 당일 투표가 몰리는 특정 지역에서 '투표용지 부족 사태'를 유발할 위험이 있습니다.
- 실제로 2026년 6.3 지방선거 당일, 전국 91개소에서 투표용지가 소진되는 사상 초유의 사태가 발생했고, 이로 인해 선관위원장과 사무총장이 사의를 표명했습니다.
- **목표:** 과거 선거(8회 지선) 데이터를 분석하여 지역별 투표 편차를 확인하고, 읍면동별 당일 투표 수요를 예측하는 모델을 구축합니다. 이를 통해 용지 부족 위험 투표소를 사전에 식별하는 데이터 기반의 의사결정 기준을 제안합니다.

> ※ 본 프로젝트는 정치적 해석을 배제하며, 데이터 기반 행정 효율화 관점에서만 접근합니다.

## 2. 기술 스택

- Language: Python 3.x
- Data Processing: Pandas, NumPy
- Visualization: Matplotlib, Seaborn
- Machine Learning: Scikit-learn, XGBoost
- Imbalanced Data Handling: SMOTE
- Model Interpretation: SHAP
- Environment: Jupyter Notebook, VS Code
- Data Source: 중앙선거관리위원회 선거통계시스템

## 3. 분석 프로세스

### Step 1. 데이터 수집 및 전처리
- 선관위 8회 지선 투표율 데이터(`xlsx`) 로드 및 다중 헤더 정제
- 당일 투표소 혼잡도와 무관한 데이터(관외사전투표, 거소투표 등) 필터링
- 읍면동 단위 분석을 위한 데이터 재구조화 (Pivot 변환)
- 선거일투표율, 사전투표율, 배정여유율 등 파생변수 생성
- 선거일투표율 기준 위험 투표소 라벨링 (평균 + 1표준편차 초과 시 위험)

### Step 2. 탐색적 데이터 분석 (EDA)
- 시도별 위험 읍면동 분포 분석 → 수도권과 농어촌 지역 모두에서 위험 패턴 확인
- 투표소 규모별 위험 발생률 분석 → 소규모·대규모 양극단에서 위험 비중이 높은 U자 패턴 확인
- 사전투표율과 선거일투표율 간 상관관계 검증 → 선관위가 전제한 반비례 관계가 뚜렷하게 나타나지 않음을 확인
- 선거일투표율 분포 분석 → 대부분 읍면동은 30% 내외이나, 우측 꼬리(극단값) 지역에서 50% 기준에 근접하거나 초과

### Step 3. 위험 투표소 분류 모델
- 불균형 데이터 문제(위험 투표소 비중 5~10%)를 해결하기 위해 SMOTE로 소수 클래스 오버샘플링 적용
- XGBoost 분류 모델로 위험 투표소 사전 식별
- 단순 정확도 대신 Recall(재현율)을 핵심 지표로 설정하고, 임계값 조정을 통해 위험 탐지율을 개선
- SHAP 분석으로 모델의 판단 근거를 해석 → 사전투표율보다 총선거인수, 지역 변수의 영향력이 더 크게 나타남
- 학습된 모델을 2022년 데이터에 적용한 백테스팅을 통해, 실제 발생한 위험 투표소의 상당수를 사전에 식별할 수 있었음을 확인

### Step 4. 선거일 투표수 회귀 예측 모델
- 읍면동별 선거일 투표수를 직접 예측하는 XGBoost 회귀 모델 구축
- 타겟 변수의 우측 비대칭 분포를 보정하기 위해 로그 변환 적용
- 예측값을 기반으로 일괄 50% 배정 방식과 모델 기반 배정 방식의 효용성을 비교하는 시뮬레이션 진행

## 4. 핵심 결론

- 일괄 50% 배정 기준은 평균적으로는 보수적인 기준이지만, 지역별 투표 수요 편차를 반영하지 못해 특정 지역에서 구조적인 부족 위험을 안고 있었습니다.
- 사전투표율 단일 변수는 선거일 투표 수요를 설명하는 데 한계가 있으며, 총선거인수와 지역 특성을 함께 고려해야 위험을 더 정확히 예측할 수 있습니다.
- 모델 기반 배정은 예산 효율성 측면에서는 우수하지만, 단독으로 최종 배정량을 결정하기에는 운영 리스크가 존재합니다. 따라서 본 모델은 기존 50% 배정 기준을 대체하는 것이 아니라, 위험 지역을 선제적으로 식별해 추가 배정을 검토하는 보조 의사결정 도구로 활용하는 것이 적절하다고 판단했습니다.
- 실제 2026년 6.3 지방선거 결과와 비교했을 때, 본 분석에서 식별한 위험 패턴이 실제 사태 발생 지역과 상당 부분 일치하는 것을 확인했습니다.

## 5. 프로젝트 한계

- 분석 단위가 실제 투표소가 아닌 읍면동 단위로, 개별 투표소의 부족 가능성을 직접 예측한 결과로 해석하기에는 한계가 있습니다.
- 2022년 선거 데이터를 기반으로 학습했기 때문에 후보 구도, 이슈 등 2026년 선거의 특수성은 반영되지 못했습니다.
- 모델의 통계적 성능과 실제 운영 환경에서의 안정성은 별개의 문제이며, 실무 적용 시에는 안전 버퍼와 운영 기준을 함께 설계해야 합니다.

## 6. 향후 개선 방향

- 실제 투표소 단위 데이터 확보를 통한 분석 단위 정교화
- 인구 구조, 접근성, 과거 시간대별 투표 흐름 등 추가 변수 반영
- 선거 전 위험 점검 지표 표준화 및 모니터링 체계 구축
