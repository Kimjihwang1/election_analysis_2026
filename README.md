# 투표소별 당일 투표 수요 예측 모델
(Election Day Vote Demand Prediction)

## 1. 프로젝트 배경 및 목적
- **배경:** 2026년 6.3 지방선거에서 선관위는 사전투표율 상승과 잔여 투표용지 관리 이슈를 이유로
본투표용지를 유권자 수의 50% 수준으로  일괄 배정하는 방식을 처음 적용했습니다.

- **문제점:** 전국 평균 사전투표율은 상승 추세이나, 읍면동 등 세부 지역별 사전투표율 편차는 매우 큽니다.
- 전체 평균에 맞춘 일괄 배정 방식은 사전투표율이 낮고 당일 투표가 몰리는 특정 지역에서 '투표용지 부족 사태'를 유발할 위험이 있습니다.

- **목표:** 과거 선거(8회 지선) 데이터를 분석하여 지역별 투표 편차를 확인하고, 읍면동별 당일 투표 수요를 예측하는 모델을 구축합니다.
- 이를 통해 용지 부족 위험 투표소를 사전에 식별하는 데이터 기반의 의사결정 기준을 제안합니다.
> ※ 본 프로젝트는 정치적 해석을 배제하며, 데이터 기반 행정 효율화 관점에서만 접근합니다.

## 2. 기술 스택
- Language: Python 3.x
- Data Processing: Pandas, NumPy
- Visualization: Matplotlib, Seaborn
- Machine Learning: Scikit-learn, XGBoost, LightGBM
- Model Interpretation: SHAP
- Environment: Jupyter Notebook, VS Code
- Data Source: 중앙선거관리위원회 선거통계시스템
